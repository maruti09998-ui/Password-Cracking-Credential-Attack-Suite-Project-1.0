Password Cracking and Credential Attack Suite

A Python toolkit for ethical password policy testing, credential security assessment, and controlled lab demonstrations.

View on GitHub Quick Start

Toolkit Scope

Dictionary Generator

Builds custom wordlists with names, dates, keyboard patterns, common passwords, and mutations.

Hash Inventory

Parses Linux shadow-style files and checks offline Windows SAM/SYSTEM hive availability.

Brute-Force Simulation

Estimates keyspace and time-to-crack for lowercase, uppercase, digit, and symbol modes.

Password Analysis

Measures complexity, entropy, predictability, severity, and recommended improvements.

Red Lab

Runs offline dictionary/mask cracking and local credential replay against explicit fixtures.

Audit Reports

Generates JSON reports with risks, weak password summaries, simulations, and policy guidance.

Quick Start

pip install -r requirements.txt python main.py --help python main.py optional-tools status python main.py report --password "Password123" -o report.json

Ethical Use

Use this project only on systems, files, and passwords you own or have explicit authorization to test. The hosted page is documentation only; the CLI and optional tools run locally in controlled lab environments.

Built for practical password policy testing and credential security education.
